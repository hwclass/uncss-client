# uncss-client

A client for uncss tool initialized by gulp

## Installation

Install via [npm][npm]:
```
npm install
```

## Usage

Use 'src' directory to store the html and css files

Run gulp process:
```
gulp
```

The eliminated css rules are selected into the `out` directory.
